Install the standard Liteplacer installation.

Backup the main LitePlacer.exe file and then replace with the file from the Release folder.

Feeder COM port is specified on the main tab with the TinyG port.

Feeder index is specified on the tape tab, as part of the existing tabulated data or as a new text field on the add tape window.

Enjoy and let me know if you have any problems relating to the feeder addition.
